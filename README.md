# NetmeTube

A youtube clone made for the purposes of learning

## The Stack

*   **Frontend:** SvelteKit (for highly responsive, server-side rendered UI states)
*   **Backend:** Express.js (handling API routing, stream endpoints, and business logic)
*   **Database:** MySQL (managing relational user profiles, video metadata, and interactions)
*   **DevOps:** Docker (fully containerized multi-service architecture for seamless deployment)
